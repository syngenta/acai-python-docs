## Requirements

* Python 3.8 or higher; [download and install Python](https://www.python.org/downloads/)
* Access to public [python registry](https://pypi.org/)


## Installation
=== "Shell"
```bash
$ pip install acai_aws
# pipenv install acai_aws
# poetry add acai_aws
```
